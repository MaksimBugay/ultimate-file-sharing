Remote Logs Viewer

Overview

Enables replacing the default JavaScript console object with
r-console, allowing you to monitor client browser logs remotely in real-time.

Main Use Cases

  • View logs from a browser on a mobile device.
  • Seamless client-side issue investigation by enabling remote access to browser logs, eliminating the need to request clients to manually send log files.

How It Works

  • Open the logs viewer website in your browser and memorize the log-session ID.
  • Open your webpage on the client side using the log-session ID as an extra query parameter: <your webpage url>?log-session=<some uuid>
  • Logs will be visible on the logs viewer page automatically with a small delayâ€”no need to refresh the page.
  • Every transfer is secured with end-to-end encryption and protected from unauthorized access, making it safe to pass sensitive data.
  • Use full-text search and log record filters to simplify analysis.
  • Reuse the same log-session ID across multiple clients to consolidate logs in one place.
